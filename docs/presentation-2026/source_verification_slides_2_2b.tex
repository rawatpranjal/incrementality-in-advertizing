\section{Key Clarifications}
%===============================================================================

\begin{enumerate}
    \item \textbf{Two Different Datasets:} Slide 2 uses the full vendor-week panel (979,290 obs, 150,075 vendors). The staggered DiD analysis uses a restricted panel (846,430 obs, 142,920 vendors) requiring cohort assignment.

    \item \textbf{ATT vs ROAS Distinction:} The ``Aggregated +0.06\%'' is the ATT coefficient from the TWFE model (log-point effect). The ROAS of 0.0904 is a separate monetary metric (return per dollar of ad spend). These should not be conflated.

    \item \textbf{Budget Tier Caveat:} The -0.10\% effect for budget vendors is \textit{not statistically significant} ($p = 0.358$).
\end{enumerate}
